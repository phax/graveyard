description: 
------------
JPdfUnit integrates PDFBox as an PDF API with the JUnit framwork for the
test of pdf documents so JPdfUnit is a high level api. Simple ready-to-use
assertions help to compare the expected data to the concrete data of the pdf
document.

homepage: 
---------
Visit the Homepage of the project @ http://jpdfunit.sourceforge.net

developers info:
----------------
The Project was developed with 
- Eclipse 3.4
	- Checkstyle Plugin 4.4.2 (http://eclipse-cs.sourceforge.net/)
- JUnit 4.x
- PDFBox 1.x

folder description: 
-------------------

		homepage/ 
		logos/ 
		testing documents/ for the examples
/lib/ ext/	the project dependencies
/src/ java/ the java sources of the project
      test/ tehe test and examples of the project
